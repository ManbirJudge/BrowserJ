The J Browser is a web browser_win developed by Manbir Singh Judge in Python programming language using PyQt5 package. You can do anything with this browser_win that you are able to do with famous web browsers like Google, FireFox, Microsoft Edge etc. except that you can not login into you google account due to Google's security reasons.

This web browser_win is very simple but not as simple as you except because you can go back and forward in history of your pages, you can go to home page which is https://www.google.com, you can also open local HTML documents and this browser_win can also tell you that web page you have opened is secure or not or is a local HTML file.

Some disadvantages:

  1. You can not login into google account.
  2. You can not open multiple tabs at ones.
  3. This is slower than other web browser_win.
